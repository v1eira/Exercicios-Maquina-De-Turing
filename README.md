### Questão 10

#### Resposta:
É uma máquina de turing que aceita apenas cadeias iniciadas com 0 seguidas de um ou mais 1, produzindo uma cadeia de saída composta somente por 1.

Exemplo:

Entrada|Saída  |
-------|-------|
0      |1
011    |111
0111111|1111111
